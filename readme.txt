The most simple and effective way to show questions and answers on your website. Create new questions, add answers, drag and drop to reorder and copy-paste the shortcode into any post/page.

== Description ==
[View demo & instructions](http://wpdarko.com/faq/)

This plugin adds a “FAQs” section to the admin panel which allows you to create questions and answers to display on your website. You can quickly add them and drag and drop them to reorder and display them anywhere with a simple shortcode. What you see is what you get, and it’s totally free.

== Installation ==

= Installation =
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find our Frequently Asked Questions plugin by WP Darko and click Install now
3. Alternatively, download the plugin and upload the contents of questions-answers.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
